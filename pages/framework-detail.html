Install and maintain a firewall

Network security controls (NSCs), such as firewalls and other network security technologies, are installed and maintained to restrict traffic between trusted and untrusted networks.

  • Firewall configuration file exists
  • Deny-all rule is present
  • Inbound traffic limited to necessary ports
