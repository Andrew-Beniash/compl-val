# Enterprise Compliance Management System (ECMS) - HTML Prototype

## Project Structure

This repository contains the HTML prototype for the ECMS application.

### Directory Organization

- **`/`**: Contains the main entry point (`index.html`).
- **`pages/`**: Contains specific page templates (e.g., Dashboard, Reports).
- **`assets/`**: Static assets.
  - `images/`: Project images.
  - `icons/`: Icon SVGs or sprites (using Google Material Symbols via CDN).
- **`css/`**: Styling files.
  - `design-system.css`: Core design tokens (Colors, Typography, Spacing).
  - `styles.css`: Component styles and layout rules.
- **`js/`**: JavaScript logic.
  - `main.js`: Global scripts.

## Design System

The project uses a custom Vanilla CSS design system based on modern standards.

### Color Palette
- **Primary**: Indigo (`#4F46E5` / `hsl(243, 75%, 59%)`) - Used for main actions and branding.
- **Secondary**: Cyan (`#06B6D4` / `hsl(189, 94%, 43%)`) - Used for accents.
- **Surface**: Slate/Gray scale for modern, clean interfaces.
- **Status Colors**: Success (Emerald), Warning (Amber), Danger (Rose).

### Typography
- **Font Family**: 'Inter', sans-serif (Google Fonts).
- **Scale**: Fluid typography using `rem`.

### Icons
- **Library**: Google Material Symbols (Outlined).

## Setup
Simply open `index.html` in any modern browser. No build step required for this prototype phase.
